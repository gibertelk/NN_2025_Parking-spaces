# NN_2025_Parking-spaces

## Задача
В данном проекте решается задача определения количества мест на парковке по фотографиям. Другими словами, перед нами стояла задача детекции мест и определении класса детектированного объекта (занято место или свободно).

## Часть 2. Выбор модели
В качестве решения была выбрана кастомная модель Faster R-CNN с backbon ResNet18. Данная модель работает с данными формата COCO, а также является легковесной, что важно из-за ограниченности ресурсов комнады.

Всего было рассмотрено 4 модели: 
1. Faster R-CNN
2. Кастомный Faster R-CNN 
3. YOLOv2
4. Кастомный YOLOv2

Среди возможных backbon рассматривались такие как:
- ResNet18
- VGG16
- EfficintNet

Для итоговой модели, показавшей лучший результат по Loss, подбиралось количество batch.
